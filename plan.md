1. 文件/图片上传与发送
支持用户上传图片、文档、PDF 等文件，并在聊天窗口中展示或让 AI 解析内容。
可以结合 api.sendMessage 或新增接口实现。
2. 聊天消息搜索 --done
在会话内支持关键词搜索历史消息，提升查找效率。
可在 ChatMessages 组件中增加搜索框和过滤逻辑。
3. 消息撤回与编辑
支持用户撤回或编辑自己发送的消息。
需要后端支持消息状态变更，前端可在消息气泡上增加操作按钮。
4. 多语言支持（i18n）--done
支持中英文等多语言切换，提升国际化能力。
可引入 react-i18next 等库，整理 UI 文本。
5. AI 角色/模型切换
支持用户选择不同的 AI 角色或模型（如助手、翻译、写作等）。
可在输入区增加下拉选择，调用不同的后端模型。
6. 消息引用与回复
支持对某条消息进行引用回复，提升讨论的上下文关联性。
UI 上可在消息上增加“回复”按钮，数据结构需支持引用关系。
7. 聊天导出与分享
支持将聊天记录导出为 PDF、Markdown、TXT 或图片，方便保存和分享。
可在会话菜单中增加导出选项。
8. 通知与推送
新消息到达时支持浏览器通知或声音提醒。
可用 Web Notification API 实现。
9. 语音输入与语音播报
支持语音输入（Speech-to-Text）和 AI 回复语音播报（Text-to-Speech）。
可用浏览器原生 API 或第三方库实现。
