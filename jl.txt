项目简介
基于 React + TypeScript 构建的管理员控制台，提供用户管理、数据分析、日志审计等功能，采用数据可视化技术直观呈现平台运营状况。

技术栈
核心框架：React 19、TypeScript 5.x
数据可视化：Recharts
UI 组件：shadcn/ui、Lucide React
样式：Tailwind CSS 4
核心功能与技术亮点
1. 数据概览仪表板
使用 Recharts 实现多维度数据可视化：
用户增长趋势图
消息量趋势柱状图
24 小时活动分布热力图
最活跃用户排行榜
统计卡片展示关键指标：总用户数、总会话数、总消息数、周增长率
2. 用户管理系统
用户列表分页展示，支持关键词搜索
用户状态管理：封禁/解封/删除 操作，支持设置封禁天数
批量操作功能：多选模式下支持批量封禁、解封、删除
用户详情弹窗：展示用户完整信息与活动记录
3. 日志审计系统
操作日志：记录管理员的所有管理操作
登录日志：记录用户登录行为（成功/失败、IP 地址、设备信息）
分页查询与时间格式化展示。




项目简介
基于 React 19 + TypeScript + Vite 构建的现代化 AI 聊天机器人前端应用，支持实时流式响应、多模态交互和多会话管理，提供桌面端和移动端自适应的精美 UI 界面。

技术栈
核心框架：React 19、TypeScript 5.x、Vite 6
UI 组件库：shadcn/ui（基于 Radix UI）、Lucide React
样式方案：Tailwind CSS 4、CSS Modules
国际化：react-i18next（中英双语支持）
其他工具：dayjs、react-markdown、remark-gfm
核心功能与技术亮点
1. 实时流式 AI 对话（SSE）
使用 Server-Sent Events (SSE) 实现打字机风格的 AI 回复，提供即时反馈体验
封装统一的 API 请求模块，支持流式读取与错误处理
实现消息自动滚动、加载动画、中断响应等交互优化
2. 多模态图片交互
支持用户上传图片进行 AI 分析（图片理解）
客户端图片压缩算法：使用 Canvas API 实现动态压缩，将图片压缩至指定大小（50KB），优化网络传输
AI 图片生成功能与历史图片浏览
3. 多会话管理
自定义 Hook 管理多个聊天会话的 CRUD 操作
支持会话重命名、删除、标题生成
4. 安全认证体系
实现完整的登录/注册流程，支持 邮箱二次认证 (2FA)
JWT Token 管理与请求拦截器封装
自定义  Hook 封装用户状态与认证逻辑
5. 其他
深色/浅色主题切换：基于 ThemeProvider 实现主题系统
响应式布局：移动端优先设计，侧边栏可折叠，自适应各种屏幕尺寸
Markdown 消息渲染、代码块高亮、消息搜索与高亮功能
消息编辑、删除、批量操作等交互功能
