## 📊 项目概览（先熟记于心）

### 技术栈
| 类别 | 技术 |
|------|------|
| **框架** | React 19, TypeScript 5.9 |
| **构建工具** | Vite 6 |
| **样式** | Tailwind CSS 4, CSS Modules |
| **UI组件库** | shadcn/ui (Radix UI) |
| **状态管理** | React Hooks (useState, useMemo, useEffect) |
| **国际化** | react-i18next |
| **图表** | Recharts |
| **HTTP请求** | Fetch API + SSE |

### 核心功能模块
1. **用户认证系统** - 注册/登录/登出/Token验证
2. **会话管理** - 多会话创建/切换/删除/重命名
3. **AI对话** - 实时流式响应(SSE)、深度思考模式
4. **多模态支持** - 图片上传与AI解析
5. **管理员后台** - 用户管理/数据统计/日志查看
6. **主题系统** - 深色/浅色模式切换
7. **国际化** - 中英文切换
